Spec notes:

  • Confirm button uses bg-gray-900 per UI §13 modal pattern (modal CTAs are never teal). Disabled until both fields valid.
  • Reason dropdown values: CUSTOMER_CANCELLED / WRONG_ORDER (matches the 2 enum values locked in the design doc).
  • Remark mandatory, 1–500 chars (trimmed). Char counter updates live; turns amber at 450+, red at 500.
  • Red warning box only appears when the OBD has a linked active challan. If no challan, the box is hidden — modal is shorter.
  • After submit: toast "OBD 8094567890 removed", board refreshes, card disappears from the column.
  • On error from API (409 stage mismatch / 403 permission / network): toast with the API error message, modal stays open, fields preserved.
