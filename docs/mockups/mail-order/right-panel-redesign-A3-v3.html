Design notes

  • Bill-to card tags: OD, CI, Bounce (blocker-red) · Bill 1/2/N (blue) · Bill Tomorrow, Cross XYZ (amber) · ✂ Bill X-Y, ⚠ Split (purple) · 7 Days, Extension, DPL (gray) · Truck Order (violet icon-only pill).
  • Ship-to card tags: Dispatch / Hold · Urgent · Challan · Slot override · ⚑ captured (when ship-to ≠ bill-to).
  • Truck Order pill: violet (bg-violet-50 / border-violet-200 / text-violet-700), icon-only with truck SVG. Distinct from blocker-red, info-gray, attention-amber, bill-blue. Operator learns the convention.
  • Match status colour rides on the code chip: gray (exact), amber (multiple), red (unmatched). Stays attached to what it modifies.
  • Lines match chip ("✓ 7/7" or "⚠ 5/7") lives in meta ribbon next to line count.
  • Ship-to amber bar: 3px left edge, only when shipToOverride=true. Reuses left-rail selected-row idiom for visual consistency.
  • Instructions strip: conditional. Three categories, dot-prefixed (amber=delivery, blue=bill, gray=notes). Only renders rows that have content. Strip collapses entirely when no remarks.
  • Footer remarks bar removed — every piece of info now above the SKU table.
  • Keyboard shortcuts all preserved: ↑↓ navigate, Space toggle, Ctrl+C copy, Ctrl+V paste SO, /-focus search, Q/W navigate orders. No new key handlers.
